1.1 Personal Information

When you create an account on Biye, we collect the following information:

  • Basic Information: Name, date of birth, gender, email address, phone number
  • Profile Information: Photographs, height, weight, marital status, education, occupation, income
  • Family Information: Father's occupation, mother's occupation, number of siblings

  • Location Information: Division, district, upazila (sub-district), current address
  • Partner Preferences: Age range, height range, skin complexion, district preferences

1.2 Authentication Information

  • Email address and password (if you register with email)

1.3 Automatically Collected Information

  • Device Information: Device type, operating system, unique device identifiers
  • Usage Data: App features accessed, time spent, search queries, profile views
  • Log Data: IP address, browser type, access times, crash logs
